# HolyDays

A very simple React.js application that helps you keep track of public holidays and other special days in any country of your choice. This repository hosts the codebase for the app's User Interface and was created as a learning project.

## Features

- Browse public holidays by country
- View holiday details and dates
- Website not responsive. Intended for desktop screens only.

## Installation & Quickstart

```bash
npm install
npm start
```

## Usage

1. Select a country from the dropdown menu
2. View holidays for the current year.
3. Click on any holiday to see additional details.

## Technologies

- React.js
- JavaScript ES6+
- CSS3
